\section{Anmerkungen}

\begin{enumerate}
    \item eine \textbf{implizite} Exception wird durch die Laufzeitumgebung geworfen, bspw. eine \code{ClassCastException}, wenn ein Typ nicht zu einem anderen Typen gecasted werden konnte (s. Abschnitt~\ref{sec:typecompatibility})
    \item eine \textbf{explizite} Exception wird durch \code{throw new [Ausnahmetyp]} eingeleitet
    \item Exceptions werden durch die \code{throws}-Klausel \textit{nach außen} (bekannt) gegeben
    \item Für einen \code{try}-Block darf es durchaus mehrere \code{catch}-Blöcke geben\footnote{
    s. ``The catch Blocks``: \url{https://docs.oracle.com/javase/tutorial/essential/exceptions/catch.html} - abgerufen 11.03.2024
    }
\end{enumerate}
